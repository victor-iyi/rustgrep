# rustgrep

A minified version of the `grep` unix command line tool.

## Usage

For case insensitive search.

```sh
$ rustgrep needle haystack.txt
```

For case sensitive search, add a `-c` flag.

```sh
$ rustgrep needle haystack.txt -c
```
